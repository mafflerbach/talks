Caching Sessions

header

  • Cache-Control: private
  • Vary: Cookie
  • Cookieless domain

Don'ts

  • Große Cookies
  • Große header

Dos

  • Statische Ressourcen ohne Set-Cookie Header
  • CSS, Bilder, JavaScript etc.

Teile deine Seite

Komponenten mit unterschiedlicher Lebensdauer

  • Eher lange dauer
    • CSS
    • JavaScript
    • Bilder
  • Kurzer dauer
    • Livetickern
    • Shoutboxen

Optimierung im kleinen

  • Ajaxifizieren
  • Cookie Aufbau schmal halten

Optimierung im großen

  • Edge Side Includes oder ESI genannt, ist eine Auszeichnungssprache
  • Proxy-Servern (z. B. Varnish, Squid and Mongrel) unterstützt
